# Tin Tech Packaging v2

This edition is a multi-page company website with a database-backed portfolio and a password-only administration portal.

## Public pages

- Home
- Company
- Capabilities
- Full product portfolio with category filters and search
- Dynamic product detail pages
- Project inquiry form
- Privacy, terms and custom 404 page

## Administration

Open `/tin-tech-packaging/admin/` and enter the shared admin password. Authorized users can create and edit categories and products, publish or unpublish listings, mark featured products, upload multiple product images, add pricing guidance, MOQ, dimensions, material, lead time and custom technical specifications, and review inquiry status.

See `BACKEND.md` for the backend architecture. Neither documentation file is included in the GitHub Pages deployment artifact.
